Kod Mimarisi

Session Management

Esnek session yönetimi ile kullanıcı tercihine göre:

# Otomatik session
async with Lisans(program_id=123) as api:
    data = await api.get_data()

# Custom session
session = aiohttp.ClientSession()
api = Lisans(program_id=123, session=session)
